## Getting Started

### Local Development
1. `pnpm install`
2. `pnpm run dev` in `packages/api` (runs on port 3001)
3. `pnpm run dev` in `packages/web` (runs on port 3000)

### Deployment
1. Configure AWS CLI: `aws configure`
2. Bootstrap: `cd infra && npx cdk bootstrap`
3. Deploy: `cd infra && npx cdk deploy`
